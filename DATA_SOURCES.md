# Data Sources

## Financial Data APIs

### yfinance
- Free stock data
- Prices, historical data, basic fundamentals
- Used for portfolio analysis and baseline metrics

### Alpha Vantage
- Financial indicators
- Technical analysis data
- Used for additional metrics and signals

### Polygon.io
- Real-time and historical market data
- More advanced (paid tiers)
- Used for scalable, real-time data (future)

---

## News APIs

### NewsAPI
- General news articles
- Easy integration
- Used for broad market context

### Finnhub
- Financial-specific news
- Market sentiment data
- Used for company-specific insights

## Data Types Used

### Structured Data
- Prices
- Volatility

### Unstructured Data
- News articles
- Reports
- Analysis documents
